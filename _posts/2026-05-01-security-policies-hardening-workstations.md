Desarrollo e implementación de un marco normativo y técnico para el endurecimiento (Hardening) de puestos de trabajo corporativos, basado en un **Plan Director de Seguridad (PDS)**.

## Fases de Implementación

1. **Definición de Políticas:** Creación de normativas para la protección del puesto de trabajo, gestión de accesos y contraseñas.
2. **Configuración Técnica:** Aplicación de políticas de grupo (GPOs) para restringir software no autorizado y asegurar configuraciones de sistema.
3. **Análisis de Trazabilidad:** Vinculación directa entre los riesgos identificados, los activos críticos y los controles técnicos aplicados.
4. **Verificación:** Elaboración de checklists de auditoría para garantizar el cumplimiento continuo de las políticas.

## Gestión de la Seguridad

Este proyecto establece una base sólida para la gobernanza de la seguridad de la información, asegurando que cada medida técnica responda a una necesidad de negocio y a un riesgo real mitigado.
